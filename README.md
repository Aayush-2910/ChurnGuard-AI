## 🎯 Overview

**ChurnGuard AI** is an enterprise-grade customer churn prediction platform that leverages advanced analytics to identify at-risk customers before they leave. Built with modern web technologies and powered by intelligent algorithms, it helps businesses reduce churn by up to **40%** and increase customer lifetime value.

```bash

# 2️⃣ Install dependencies
```

## 🔌 API Documentation

### 📡 Prediction Endpoint

**POST** `/predict`

Predicts customer churn probability based on 19 input features.

#### 📥 Request Body
```json
{
  "gender": "Male",
  "SeniorCitizen": 0,
  "Partner": "Yes",
  "Dependents": "No",
  "tenure": 12,
  "PhoneService": "Yes",
  "MultipleLines": "No",
  "InternetService": "Fiber optic",
  "OnlineSecurity": "No",
  "OnlineBackup": "Yes",
  "DeviceProtection": "No",
  "TechSupport": "No",
  "StreamingTV": "Yes",
  "StreamingMovies": "Yes",
  "Contract": "Month-to-month",
  "PaperlessBilling": "Yes",
  "PaymentMethod": "Electronic check",
  "MonthlyCharges": 85.50,
  "TotalCharges": 1020.00
}
```

#### 📤 Response
```json
{
  "success": true,
  "prediction": 1,
  "churn_probability": 78.5,
  "risk_level": "High"
}
```

#### 📊 Status Codes
- `200` ✅ Success
- `400` ❌ Bad Request
- `500` 🔥 Server Error

## 🔧 Configuration

### 🌐 Network Access

Access from other devices on your network:

```bash
# 1️⃣ Find your IP address
ipconfig          # Windows
ifconfig          # Mac/Linux

# 2️⃣ Access using
http://YOUR_IP:5000
```

### ⚙️ Environment Variables

Create a `.env` file in the root directory:

```env
FLASK_APP=app.py
FLASK_ENV=development
SECRET_KEY=your-secret-key-here
DEBUG=True
HOST=0.0.0.0
PORT=5000
```

### 🔌 Port Configuration

Change the default port if needed:

```bash
# Use custom port
python app.py --port 5001
```

## 🐛 Troubleshooting

<details>
<summary>🔴 Server won't start</summary>

```bash
# Check if port 5000 is already in use
netstat -ano | findstr :5000

# Kill the process or use different port
python app.py --port 5001
```
</details>

<details>
<summary>🔴 CSS/JS not loading</summary>

- Clear browser cache: `Ctrl + Shift + R`
- Check browser console for errors
- Verify file paths in HTML files
- Ensure static folder structure is correct
</details>

<details>
<summary>🔴 Prediction errors</summary>

- Ensure all 19 fields are filled
- Check data types (numbers vs strings)
- Verify backend is running
- Check browser console for API errors
</details>

<details>
<summary>🔴 Import errors</summary>

```bash
# Reinstall dependencies
pip install -r requirements.txt --force-reinstall

# Or use virtual environment
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```
</details>
